# UniText Converter

Simple client-side tool to extract text from images (OCR), PowerPoint files, and PDFs. Everything runs in your browser. No server needed for PowerPoint and text-based PDFs – your files stay private.

Dark monochrome design, batch upload, collapsible results, easy copy options. Just one single HTML file.

## Live Demo

https://atosacik.github.io/UniText-Converter/

## How to Use

1. Open the page.
2. Choose Image / PowerPoint / PDF.
3. Drag & drop or pick your files.
4. For images (or scanned PDFs): enter a free OCR.space API key → get one here: https://ocr.space/ocrapi (25,000 free requests/month).
5. Click "Convert to Text".
6. Copy single blocks, whole files, or everything.

PowerPoint and normal PDFs work 100% offline/client-side.
